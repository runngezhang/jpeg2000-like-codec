DESCRIPTION 

 UpSample -- Upsampling operator
  Usage
    u = UpSample(d[,s]) 
  Inputs
    d   1-d signal, of length n
    s   upsampling scale, default = 2
  Outputs
    u   1-d signal, of length s*n with zeros
        interpolating alternate samples
        u(s*i-1) = d(i), i=1,...,n
